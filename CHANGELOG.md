# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

### Added
- Initial release v0.1.0
  - Magic byte detection (PE, ELF, PNG, JPEG, PDF, ZIP, etc.)
  - Data inspector for multi-format value decoding
  - Pattern language parser for custom binary structures
  - Entropy analysis for detecting packed/encrypted regions
  - Binary search (strings, hex patterns, integers)
  - Export to JSON, CSV, YAML
  - CLI with Rich terminal output
